# OMNE Open Node Project
OMNE open source node

## Setup Instructions

### Prerequisites

- Docker
- Docker Compose

### Cloning the Repository

1. Clone the repository to your local machine:

```sh
git clone https://github.com/your-username/omne-open-node.git
cd omne-open-node
```

**### Running the Project**

1. Build and start the Docker containers using Docker Compose:

2. Verify that the containers are running by checking the status:

You should see your node containers running.

### Verifying Class Hashes

The node will automatically verify the class hashes against the stored hashes fetched from the OMNE Oracle. If the verification fails, the node will log an error and exit.

### Additional Notes

- Ensure your Docker setup is correct and the Docker Compose file is properly configured. (do not edit this file)

- The project includes an API for fetching class hashes to use in the verification process. 

### Classes Verified

- OMC (the class for OMNE Coin)
- C_UTILS (class with various cryptography implementations and tools)
- DynamicFeeCalculartor (class responsible for calulating tx fees)
- Verifier (class with varying components used to verify data at differnt stages of the consensus protocol)
