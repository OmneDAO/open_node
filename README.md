# OMNE Open Node Project
OMNE open source node

## Setup Instructions

### Prerequisites

- Docker
- Docker Compose

### Environment Variables

Before running the project, ensure you have set the following environment variables:

```sh
export STEWARD_ADDRESS=<your_omne_address>
```

### Running the Node

```sh
docker build -t omne-node:latest .
```

```sh
docker run -it --name omne-node omne-node:latest
```

### Verifying Class Hashes

The node will automatically verify the class hashes against the stored hashes fetched from the OMNE Oracle. If the verification fails, the node will log an error and exit.

### Additional Notes

- Ensure your Docker setup is correct and the Docker Compose file is properly configured. (do not edit this file)

- The project includes an API for fetching class hashes to use in the verification process.
